# Skills

个人自创和整理的 OpenClaw agent skills。

## 技能列表

| 技能 | 说明 |
|---|---|
| grilling | Grill 用户关于方案/设计/决策，逐分支遍历决策树 |
| lifelog | 生活记录自动化，自动识别日期并记录到 Notion |
| file-share | 通过公网 URL 分享服务器文件 |
| paper-downloader | 批量下载学术论文并规范重命名 |
| unified-self-improving | 统一自我进化系统，结构化日志与模式检测 |
| skillhub-preference | SkillHub 优先于 ClawHub 的偏好配置 |
| talk-normal | 简洁直接的沟通风格指令 |
| iclr-paper-recommender | ICLR 论文个性化推荐 |
| latex2md | LaTeX 转 Markdown |
| xiaohongshu-search-summary | 小红书搜索与内容整理 |
| xiaohongshu-skills | 小红书自动化技能集合 |
| zotero-iclr-recommender | 基于 Zotero 的 ICLR 论文推荐 |
