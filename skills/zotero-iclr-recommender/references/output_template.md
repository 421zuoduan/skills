## B. 推荐 Top 20 论文

### 格式

```
[推荐等级] S / A / B
[论文标题]
[作者]
[接收状态]
[主题标签]
[推荐分数] 0-100

[为什么推荐]
用 3-6 句话说明：
- 它和您 Zotero 库中的哪些方向最相关
- 它像是在延续或改进哪些您已经关注的方法
- 它的创新点在哪里
- 您应该优先看哪一部分（问题定义 / 方法 / 实验 / 理论）

[关联到我库中的已有论文]
列出 3-8 篇最相关 Zotero 条目，并简述关联理由

[阅读建议]
精读 / 略读 / 先收藏
```

## E. 使用说明

1. 确保 `data/zotero_papers.json` 存在（运行 fetch_zotero.py）
2. 确保 `data/iclr2026_search.json` 存在（运行 fetch_iclr.py）
3. 运行 `python scripts/match_and_rank.py` 生成初步推荐
4. 使用大模型进一步细化分析和推荐理由
